# Release v0.1.0-alpha 🚀

The first public "Pro" release of the Spotify Taskbar Widget!

## What's New? ✨
- **Ultra-Slim 30px Profile**: Perfectly calibrated for the Windows 11 taskbar.
- **Dynamic Accent Colors**: The UI automatically shifts colors to match your current song's album art.
- **System Tray Integration**: Close the window to keep it running in the background.
- **Dynamic Taskbar Detection**: Automatically snaps to the edge of the screen on any resolution or DPI scaling.
- **Global Media Controls**: Full support for play, pause, next, and previous keys.

## Assets 📦
- `Spotify Taskbar Widget_x64-setup.exe` (Recommended)
- `Spotify Taskbar Widget_x64_en-US.msi`

## How to Install? 🛠️
1. Download the `.exe` installer.
2. Run the setup.
3. Authenticate with Spotify and enjoy!
